OS Differences

Windows

  • Very high software compatibility, most PC users use Windows so most software works for Windows. Great for PC gaming.
  • Easy operating system to learn and customize.
  • Works well with additional peripherals. (like speakers or cameras)
  • Less Secure.
  • Can easily upgrade or build your own system using Windows or Linux.

Mac OS

  • Simple and easy to learn user interface.
  • Uses gestures and touchpad navigation. Great at multitasking.
  • Usually more secure than Windows. Viruses and malware is less common.
  • Can be very expensive.
  • More difficult to build from scratch or upgrade hardware.
  • Integrates with other Apple products including the AppStore and GarageBand, for example.

Linux

  • Free and Open Source, this means you can download the system for free but the hardware will have to be purchased seperately.
  • Multiple options of operating systems called distros. Can pick and choose the look and feel of your system by default.
  • Very secure, popular OS choice for servers.
  • Works with the least amount of software compared to the other two systems, but linux compatibility is increasing over time.
  • Compatible with older computer hardware.
